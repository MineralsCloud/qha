Convert :math:`f(V, T)` to :math:`f(P, T)`
*********************************************

This module will convert a function :math:`f` of volume (:math:`V`) and temperature (:math:`T`),
e.g., :math:`F(V, T)`, :math:`G(V, T)`, :math:`C_V(V, T)`, etc.,
to the function of pressure and temperature, i.e., to :math:`F(P, T)`, :math:`G(P, T)`, and :math:`C_V(P, T)`, etc.,
where one :math:`P` corresponds to one :math:`V`.
