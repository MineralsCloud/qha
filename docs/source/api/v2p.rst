Convert :math:`f(T, V)` to :math:`f(T,P)`
*********************************************

This module will convert a function :math:`f` of volume (:math:`V`) and temperature (:math:`T`),
e.g., convert :math:`G(T, V)` to the function of temperature and pressure, i.e., to :math:`G(T, P)`.
