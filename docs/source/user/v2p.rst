Parameterize :math:`(V, T)` to :math:`(P, T)`
*********************************************

This module will parameterize a function of volume and temperature, :math:`F(V, T)`, :math:`G(V, T)`, :math:`C_v(V, T)`, etc.,
to the function of pressure and temperature, i.e., to :math:`F(P, T)`, :math:`G(P, T)`, and :math:`C_v(P, T)`, etc., where
:math:`P` corresponds to the :math:`V`.
