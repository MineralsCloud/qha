1.9. Thermodynamic properties¶

This module is used for calculating various thermodynamic properties using Helmholtz free energy.

qha.thermodynamics.pressure_tv(v_vector, free_energies)[source]¶

Equation used to calculate pressure: \(P = - \left ( \frac{ \partial F }{ \partial V } \right )_T\)

Parameters:
  • v_vector (array(float64, 1d, A)) – fine volumes vector
  • free_energies (array(float64, 2d, A)) – \(F(T,V)\)
Return type:

array(float64, 2d, A)
Returns:

pressure, \(P(T,V)\)

qha.thermodynamics.thermodynamic_potentials(ts, v_vector, free_energies, p_tv)[source]¶

Calculate \(H\), \(U\), and \(G\) on \((T,V)\) grid from \(F\);

\(U = F + TS, H = U + PV, G = F + PV\)

input: nt, and F, which is Helmholtz Free Energy, are needed.

Parameters:
  • free_energies (array(float64, 2d, A)) – \(F(T,V)\)
  • p_tv (array(float64, 2d, A)) – pressure, \(P(T,V)\)
  • ts (array(float64, 1d, A)) – temperature vector
  • v_vector (array(float64, 1d, A)) – fine volumes vector
Returns:

\(U\), \(H\), \(G\) on \((T,V)\) grid
