1.7. Interpolate on a dense volume mesh¶

This module contains a grid interpolation algorithm that integrates previous Birch–Murnaghan equation of state fitting, to get Helmholtz free energy on a denser volume mesh.

qha.grid_interpolation.calc_eulerian_strain(v0, v)¶

Calculate Eulerian strain (\(f\)) of a given volume vector v regarding to a reference v0, where

\[f = \frac{ 1 }{ 2 } \bigg( \Big( \frac{ V_0 }{ V }^{2/3} \Big) -1 \bigg).\]
Parameters:
  • v0 – The volume set as reference for Eulerian strain calculation.
  • v – The volume to be calculated its strain W.R.T. v0.
Returns:

Calculated Eulerian strain.
qha.grid_interpolation.from_eulerian_strain(v0, f)¶

Calculate corresponding volume \(V\) from given Eulerian strain (f) and reference volume v0. It is the inverse of calc_eulerian_strain function, i.e.,

\[V = V_0 (2 f + 1)^{-3/2}.\]
Parameters:
  • v0 – The volume set as reference for volume calculation.
  • f – Eulerian strain for \(V\) W.R.T. \(V_0\).
Returns:

Calculated volume \(V\).
qha.grid_interpolation.interpolate_volumes(in_volumes, out_volumes_num, ratio)[source]¶

Interpolate volumes on input, with ratio given. For Eulerian strain, the larger the strain, the smaller the volume. So large volume corresponds to small strain.

Parameters:
  • in_volumes – The input sparse 1D array of volumes.
  • out_volumes_num – Number of output volumes. It should be larger than the number of input volumes.
  • ratio – The ratio of the largest volume used in fitting W.R.T. the largest input volumes.
Returns:

The interpolated strains in a finer grid, and corresponding volumes.
class qha.grid_interpolation.RefineGrid(desired_p_min, nv, order=3)[source]¶

A class that will do the Birch–Murnaghan finite-strain EoS fitting, and evaluate the fitted function on a denser volume grid.

Parameters:
  • desired_p_min (float) – The desired minimum pressure to calculate for further steps.
  • nv (int) – The number of volumes on a denser grid.
  • order (Optional[int]) – The order of the Birch–Murnaghan finite-strain EoS fitting.
approach_to_best_ratio(volumes, free_energies, initial_ratio)[source]¶

Trying to find the best volume grids based on an a very large volume grids.

Parameters:
  • volumes (array(float64, 1d, A)) – Volumes of these calculations were perform (sparse).
  • free_energies (array(float64, 1d, A)) – Free energies at the highest temperature (sparse).
  • initial_ratio (float) – Initial ratio, a guess value, which can be set to a very large number.
Return type:

float
Returns:

The suitable ratio for further calculation.
refine_grid(volumes, free_energies, ratio=None)[source]¶

Get the appropriate volume grid for interpolation. Avoid to use a too large volume grid to obtain data, which might lose accuracy.

Parameters:
  • free_energies (array(float64, 2d, A)) – Calculated Helmholtz Free Energies for input volumes (sparse).
  • volumes (array(float64, 1d, A)) – Volumes of these calculations were perform (sparse).
  • ratio (Optional[float]) – This ratio is used to get a larger volume grid
Return type:

Tuple[array(float64, 1d, A), array(float64, 2d, A), float]
Returns:

volume, Helmholtz free energy at a denser vector, and the ratio used in this calculation
