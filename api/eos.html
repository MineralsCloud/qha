1.11. Equation of states¶

Nearly all the functions in this module have been speeduped by Numba package, which means they are fast, but all the arguments must be of type float.

qha.eos.parabola(v, a, b, c)[source]¶

This function is used to fit the data in parabola polynomial: \(c v^2 + b v + a\). A second order function seems to be sufficient, and guarantees a single minimum.

Parameters:
  • v (float) – Volume at which free energy is to be calculated.
  • a (float) – Constant.
  • b (float) – Coefficient of the linear term.
  • c (float) – Coefficient of the 2nd-order term.
Return type:

float
Returns:

The calculated free energy at volume v.
qha.eos.birch(v, b0, bp0, v0, f0=0)[source]¶

Calculate free energy at volume v, according to Birch EOS. [1]

[1]From “Intermetallic compounds: Principles and Practice, Vol. I: Principles Chapter 9 pages 195-210 by M. Mehl. B. Klein, D. Papaconstantopoulos”, case where \(n=0\).
Parameters:
  • v (float) – Volume at which free energy is to be calculated.
  • b0 (float) – Bulk modulus at zero-pressure, as a reference parameter for the EOS.
  • bp0 (float) – Bulk modulus’s first order derivative W.R.T pressure at zero-pressure, as a reference parameter for the EOS.
  • v0 (float) – Zero-pressure volume for the system.
  • f0 (Optional[float]) – The baseline of free energy, defaults to be \(0\).
Return type:

float
Returns:

The calculated free energy at volume v.

class qha.eos.EOS(b0, bp0, v0, f0=0)[source]¶

Bases: object

An abstract base class for equations of states. The classes Birch, Murnaghan, BirchMurnaghan3rd, PourierTarantola, and Vinet are all subclasses of this class. Two of its public methods, free_energy_at and pressure_at are not implemented here, so they must be implemented in its subclasses.

Parameters:
  • b0 (float) – Bulk modulus at zero-pressure, as a reference parameter for the EOS.
  • bp0 (float) – Bulk modulus’s first order derivative W.R.T pressure at zero-pressure, as a reference parameter for the EOS.
  • v0 (float) – Zero-pressure volume for the system.
  • f0 (Optional[float]) – The baseline of free energy, defaults to be \(0\).
